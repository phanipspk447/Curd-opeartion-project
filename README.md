Project Description

This is a simple To-Do List web application that allows users to add and remove tasks.
It is built using basic web technologies like HTML, CSS, and JavaScript.

Features

Add new tasks

Remove tasks

Simple and clean user interface

Runs directly in the browser

Technologies Used

HTML

CSS

JavaScript

Project Structure
todo-list

├── index.html
├── style.css
└── script.js
How to Run

Download or clone the repository.

Open the project folder.

Open index.html in your browser.
